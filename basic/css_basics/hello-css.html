Estilo a las listas

Puedes estilizar listas no ordenas de la siguiente forma:

  • disc
  • circle
  • square

Y puedes enumerar listas de la siguiente forma:

  1. decimal
  2. lower-roman
  3. upper-roman
  4. lower-alpha
  5. upper-alpha
  6. (and many more!)
